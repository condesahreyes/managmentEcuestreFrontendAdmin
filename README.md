# Frontend Admin - Centro Ecuestre

Panel administrativo para gestión del centro ecuestre.

## Características

- ✅ Agenda general con calendario interactivo
- ✅ Gestión de alumnos (bloquear/activar)
- ✅ Gestión de caballos (cambiar estado)
- ✅ Gestión de planes
- ✅ Estadísticas y ocupación

## Instalación

```bash
npm install
```

## Configuración

Crea un archivo `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:3001/api
```

## Ejecutar

```bash
# Desarrollo
npm run dev

# Producción
npm run build
npm start
```

La aplicación estará disponible en `http://localhost:3002`
